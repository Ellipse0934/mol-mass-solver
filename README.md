# Molecular mass solver
Example:
```
$ git clone https://github.com/farkon00/mol-mass-solver
$ cd mol-mass-solver
$ python3 main.py # Use ctrl + C to exit
H2O
18
Ca(OH)2  
97
^C
$ python3 main.py -v # Verbose
CO2
1 * 12 + 2 * 16 = 44
^C
$ rlwrap python3 main.py # To add history and navigation with arrows, rlwrap is an external package
```
